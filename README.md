
#FASTAPI Product API
AREST API built with fastapi and pydantic with full CRUD operations and data validation.
###FEATURES
-create,read,update,delete products
-pydantic validation with Field constraints
-Custom field validators for business logic
-Discount cannot exceed product price
-Quantity and price must be greater than 0
-Minimum name length validation
###Tech Stack
Python,FastAPI,Pydantic
### what i learned
-Field constraints (gt,ge,min_length)
-Field_validator for cross-field validation
-Defensive None checking before comparisons
